5. Explain the concept of Scope in JavaScript (Global, Function, Block).
   Ans: In JavaScript scope means the accessibility of variable, functions and objects across the codebase. JavaScript has 3 types of scopes they are Global Scope, Function & Block scope.
   Global Scope: These can be accessed any where across the codebase;
   Function Scope: The can only be accessible within the function where they are defined, they can't be accessed outside the functional scope;
   Block Scope: These only be accessible within the block where they are declared.
